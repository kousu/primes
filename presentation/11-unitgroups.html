Unit Groups

Def'n: U(n) is the group of units of modulo n. This is the set of all congruence-classes (~= integers, whatever) in mod n where all members are coprime to n.

e.g. U(2) = [1]
U(3) = [1, 2]

Some things we know about these: the size of the unit group is ephi(n) where ephi is the euler-phi function. If n is prime, ephi(n) = n-1 because everything except n (aka 0) itself is coprime to n. If we have n factored into primes, then n=p^i * q^j * r^k, then ephi(n) = (p^i - p^(i-1))*(q^j - q^(j-1))*(r^k - r^(k-1))
