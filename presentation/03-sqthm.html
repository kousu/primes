The Square-Factor Theorem (--> Math 135)

Thm: if a number I is composite, it will have at least one factor < sqrt(I)

Proof: Let I be composite. That means it can be written as I = fq for some factors f and q.

Suppose both f and q are larger than the square root. Then f > sqrt(I) => f*q > sqrt(I)*sqrt(I) = I. But f*q = I so this is impossible.

QED

This is a very key result for prime sieving! Turning it around, it means that we know a number will have been sieved out already if it is less than the *square* of the current prime. We don't have to even look at it again.

e.g. You need not sieve 35=7*5 because 35 < 49. On 29 you need not sieve 754=2*13*29 and 493=17*29 because 29^2 = 841, larger.
